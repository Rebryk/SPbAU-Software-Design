# Roguelike
###Kill the monsters! Steal the threasure!

## Commands
`w` - go up <br>
`s` - go down <br>
`a` - go left <br>
`d` - go right <br>
`activate <i>` - activate the i-th artifact <br>
`deactivate <i>` - deactivate the i-th artifact <br>

## Monsters
* Goblin
* Scavenger

## Artifacts
* Sword 
* Shield
* Medicine chest
